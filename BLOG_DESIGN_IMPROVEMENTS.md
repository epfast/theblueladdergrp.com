## ðŸ–¼ï¸ Image Support

### Featured Image
- Placed at top of blog post
- Full-width hero placement
- 300px (mobile) to 500px (desktop) height
- Gradient overlay
- Specified in frontmatter: `image: /your-image.jpg`

### Inline Images
- Add anywhere in your markdown content
- Syntax: `![Alt text](/image-path.jpg)`
- Automatic styling applied:
  - Rounded corners (xl)
  - Box shadow (lg)
  - Border for definition
  - Proper spacing (8 units vertical margin)

### Image Best Practices
1. Place images in `/public/` directory
2. Use descriptive filenames
3. Recommended sizes:
   - Featured: 1200x600px (2:1 ratio)
   - Inline: 800-1200px wide
4. Use WebP format for best performance
